\# Информационная система учёта заявок на обслуживание техники



\## Описание



Программа для сервисного центра. Позволяет вести учёт клиентов, техники и заявок на ремонт.



\## Функции



\- Добавление, редактирование, удаление и просмотр клиентов

\- Добавление, редактирование, удаление и просмотр техники

\- Создание заявок с выбором клиента, техники и мастера

\- Изменение статуса заявки: Принята → Обработка → Выполнение → Готова → Закрыта

\- При первом запуске автоматически добавляются сотрудники и мастера



\## Статусы заявок



| Статус | Описание |

| :--- | :--- |

| Принята | Заявка создана |

| Обработка | Идёт диагностика |

| Выполнение | Идёт ремонт |

| Готова | Ремонт завершён |

| Закрыта | Техника выдана клиенту |



\## Установка и запуск



1\. Установите Python 3.7 или выше

2\. Склонируйте репозиторий:

&#x20;  git clone https://github.com/ваш-логин/service\_center.git

3\. Перейдите в папку проекта:

&#x20;  cd service\_center

4\. Запустите программу:

&#x20;  python main.py
