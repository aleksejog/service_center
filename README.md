\# Информационная система учёта заявок на обслуживание техники



\## Описание проекта



Программа предназначена для автоматизации учёта заявок на ремонт техники

в сервисном центре. Позволяет вести базу клиентов, учёт техники и заявок

на обслуживание.



\## Функциональность



\- Управление клиентами (добавление, редактирование, удаление, просмотр)

\- Управление техникой (добавление, редактирование, удаление, просмотр)

\- Управление заявками (создание, изменение статуса, удаление, просмотр)

\- Предустановленные сотрудники и мастера



\## Статусы заявок



| Статус | Описание |

| :--- | :--- |

| Принята | Заявка зарегистрирована в системе |

| Обработка | Проводится диагностика техники |

| Выполнение | Выполняется ремонт |

| Готова | Ремонт завершён, ожидание клиента |

| Закрыта | Техника выдана клиенту |
